# Data Channels Command Line Interface

The Data Channels CLI is meant to provide a slim wrapper around the SDK and API services for testing, debugging, and data loading purposes.

## Install

Build and link

```bash
$ npm install
$ npm build
$ sudo npm link
```

## Usage

### Help

```bash
# List all commands
$ dchan --help

# Shorthand
$ dchan -h

```
